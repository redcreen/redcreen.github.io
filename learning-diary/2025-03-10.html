Mac Air 2018 安装踩坑

在 Mac Air 2018 款上安装 OpenClaw（龙虾），先升级了系统，安装和启动都成功了。

问题：一直出现 LLM request timeout，请求发出去 2 秒后就会超时跳出。

尝试了各种解决办法：

  • 修改 timeout 参数
  • 调整龙虾自身的配置
  • 设置系统变量
  • 修改 HTTP 请求的默认值

中间猜测是大模型的适配问题，也试了 Qwen2.5、MiniMax，最终使用 Moonshot，还是不行。

结论：看起来是 Mac Air 老版本兼容问题，等空了再去尝试解决。
安装问题 超时 Mac Air 2018 待解决

Windows 安装成功

当天晚上，在 Windows 上按照官网推荐的方式安装了 OpenClaw。

✅ 使用 Moonshot API Key，一次性就成功了！

对比 Mac 上的折腾，Windows 上的安装体验非常顺畅。

🧪 实验：携程机票查询自动化

同天晚上，尝试让龙虾帮忙做一个打开网页、去携程上查询机票的功能。

要求：不是用 URL 直接定位的方式，而是让龙虾自己学习去页面填写内容。

龙虾使用的是内置浏览器去尝试填写表单。在选择"单程"部分时，他总是搞不定，一直自己尝试各种方案。

结果：虽然最终失败了（后来让他停掉了），但他自我学习的能力真的很赞！

看着龙虾不断尝试、调整策略的过程，虽然没成功，但让我对它的自主学习能力印象深刻。

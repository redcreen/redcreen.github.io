什么是 OpenClaw？

OpenClaw 是一个开源的多代理 AI 系统，让你可以在单一网关中运行多个独立的 AI 助手（Agent）。每个 Agent 都有自己的：

  • 🧠 独立人格 - 通过 SOUL.md / AGENTS.md 定义
  • 💾 独立工作区 - 文件、记忆、会话完全隔离
  • 🔐 独立认证 - 各 Agent 有自己的 API 密钥
  • 🛠️ 独立工具 - 可为不同 Agent 配置不同工具权限

🌐 PowerShell 代理配置指南

在 PowerShell 中使用 v2raYN 代理的完整配置教程

在 PowerShell 里使用 v2raYN 代理，关键在于理解两种不同的"代理"模式，以及如何让 PowerShell "听" v2raYN 的话。

简单来说，v2raYN 本身是一个代理客户端，它会在你电脑的本地地址（127.0.0.1）上开启特定的端口（比如 10808 和 10809）来提供代理服务。要让 PowerShell 使用这个服务，你需要手动告诉它通过哪个端口来连接外部网络。

🧐 理解两种"代理"模式

这里有两个容易混淆的概念，搞清楚它们，问题就解决了一半：

1. v2raYN 的"系统代理"模式

这是 v2raYN 的一个功能开关。当你开启它时，v2raYN 会自动修改 Windows 的系统设置，让大部分图形界面程序（如浏览器）的流量通过代理。

⚠️ 注意：这个操作并不会自动配置 PowerShell、CMD 这类命令行工具。

2. PowerShell 的"环境变量代理"

PowerShell 以及很多命令行工具（如 git、npm、pip）会通过读取名为 http_proxy 和 https_proxy 的环境变量来决定是否使用代理以及使用的地址。

💡 关键：这是它们自己独立的"开关"，需要单独配置。

所以，即使你在 v2raYN 里开启了"系统代理"，PowerShell 也可能因为不知道代理的地址而无法使用。你需要手动在 PowerShell 里设置这两个环境变量。

🛠️ 如何为 PowerShell 配置代理

1. 找到你的 v2raYN 代理端口

首先，你需要确认 v2raYN 正在使用的端口号。通常：

  • HTTP 代理端口：默认为 10809
  • SOCKS5 代理端口：默认为 10808

为了确保无误，你可以查看 v2raYN 的参数设置，或者直接查看其核心配置文件夹中的 config.json 文件来确认端口。

2. 在 PowerShell 中设置环境变量（临时生效）

打开你的 PowerShell 窗口，复制并粘贴以下命令后按回车。这种方式设置的代理只在当前窗口有效，窗口关闭后失效，非常适合临时使用。

$env:http_proxy="http://127.0.0.1:10809"
$env:https_proxy="http://127.0.0.1:10809"
💡 提示：如果你的 v2raYN 配置的 HTTP 端口不是 10809，请将命令中的端口号替换成你自己的。

3. 验证代理是否生效

设置完成后，可以在同一个 PowerShell 窗口中运行以下命令来测试：

curl.exe https://ipinfo.io

如果返回的 IP 地址是你代理服务器的 IP，而不是你本地的宽带 IP，那就说明配置成功了。你也可以用 Invoke-WebRequest 命令测试：

Invoke-WebRequest -Uri https://ipinfo.io -UseBasicParsing
⚠️ 注意：不要用 ping 命令来测试，因为它使用的是 ICMP 协议，不走 HTTP/SOCKS 代理，所以永远测试不出来。

4. （可选）让配置永久生效

如果你希望每次打开 PowerShell 都能自动使用代理，可以将代理信息添加到系统的环境变量里。

方法一：PowerShell 命令（推荐）

以管理员身份打开 PowerShell，运行以下命令。设置后，需要重启 PowerShell 窗口才能生效。

[Environment]::SetEnvironmentVariable("HTTP_PROXY", "http://127.0.0.1:10809", "User")
[Environment]::SetEnvironmentVariable("HTTPS_PROXY", "http://127.0.0.1:10809", "User")

方法二：图形界面

  1. 按下 Win + R 键，输入 sysdm.cpl 并回车。
  2. 点击"高级"选项卡 → "环境变量"。
  3. 在"用户变量"中，点击"新建"，分别创建变量名为 HTTP_PROXY 和 HTTPS_PROXY，变量值都设为 http://127.0.0.1:10809。

👋 如何取消代理设置

当你不再需要使用代理时，记得及时取消，以免影响访问内网或本地服务。

临时取消（当前窗口）

Remove-Item Env:http_proxy
Remove-Item Env:https_proxy

永久取消

通过系统环境变量设置的，需要重新打开环境变量编辑界面，手动删除你之前添加的 HTTP_PROXY 和 HTTPS_PROXY 变量。

⭐ 最受欢迎的 Skills

ClawHub 官方推荐的热门 Skills（实时截图更新）

以下是 clawhub.ai/skills 上的热门 Skills，按官方排序：

1

📝 self-improving-agent

捕获学习经验、错误和修正，实现持续改进。自动记录错误、用户纠正和最佳实践。

npx clawhub install self-improving-agent
2

🎯 First Skill

帮助用户发现和入门 skills，当用户问"How do I..."时提供引导。

npx clawhub install first-skill
3

📄 Summarize

从 URL、播客和本地文件中总结或提取文本/转录内容。

npx clawhub install summarize
4

🌐 Agent Browser

AI 代理的浏览器自动化 CLI，支持网页交互、表单填写、截图等。

npx clawhub install agent-browser
5

📧 GWS (Google Workspace)

Google Workspace CLI，支持 Gmail、Calendar、Drive、Contacts、Sheets、Docs 等。

npx clawhub install gws
6

🐙 GitHub

使用 `gh` CLI 与 GitHub 交互，支持仓库管理、Issue、PR 等。

npx clawhub install github
7

🧠 ontology

结构化代理记忆和可组合技能的类型知识图谱。

npx clawhub install ontology
8

🚀 Proactive Agent

将 AI 代理从被动响应者转变为主动合作伙伴，预测需求并主动行动。

npx clawhub install proactive-agent
9

🌤️ Weather

通过 wttr.in 或 Open-Meteo 获取当前天气和预报，无需 API Key。

npx clawhub install weather
10

🔍 Skill Vector

技能的向量内存，支持语义搜索、少样本匹配、权限检查和子技能模式。

npx clawhub install skill-vector

📌 如何安装 Skills

使用 ClawHub CLI 安装：

# 搜索 skills
npx clawhub search <关键词>

# 安装 skill
npx clawhub install <skill-name>

💡 提示

访问 clawhub.ai/skills 发现更多实用 skills，目前共有 2770+ 个 skills 可供选择。

⭐ 最受欢迎的 Skills

ClawHub 上热门的 Skills，按搜索热度排序

🐙 Github

🔥 热门

GitHub 操作集成，支持仓库管理、Issue 处理、PR 管理等完整 GitHub 工作流。

npx clawhub install github

🚀 Vercel Deploy

🔥 热门

一键部署到 Vercel，支持静态网站、Next.js、React 等框架的自动部署。

npx clawhub install vercel-deploy

🌐 Web Deploy GitHub Pages

🔥 热门

自动部署到 GitHub Pages，适合静态网站、文档站点的托管。

npx clawhub install web-deploy-github

🤖 OpenClaw GitHub Assistant

OpenClaw 专用的 GitHub 助手，提供更深入的 GitHub 集成功能。

npx clawhub install openclaw-github-assistant

📊 Kanban

构建多项目看板系统，支持任务管理、进度追踪和团队协作。

npx clawhub install kanban

🔍 GitHub Search

强大的 GitHub 搜索功能，快速查找仓库、代码、Issue 和 PR。

npx clawhub install github-search

🎫 GitHub Issue Resolver

自动分析和解决 GitHub Issue，提供修复建议和代码实现。

npx clawhub install github-issue-resolver

⚡ GitHub Actions Generator

自动生成 GitHub Actions 工作流，支持 CI/CD、测试、部署等场景。

npx clawhub install github-actions-generator

🚀 Railway Deploy

一键部署到 Railway 平台，支持多种语言和框架的云部署。

npx clawhub install railway-deploy

🔴 Red Team

对抗性多 Agent 辩论引擎，用于压力测试和方案验证。

npx clawhub install red-team

📌 如何安装这些 Skills

使用 ClawHub 安装：

# 搜索 skills
npx clawhub search <关键词>

# 安装 skill
npx clawhub install <skill-name>

💡 提示

ClawHub 上的 skills 会不断更新，建议定期访问 clawhub.com 发现更多实用 skills。

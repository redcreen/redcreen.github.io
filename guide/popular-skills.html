📌 如何安装这些 Skills

使用 find-skills 或直接访问 GitHub 仓库：

# 使用 find-skills 搜索
npx skills find <关键词>

# 或直接安装
npx skills add <owner/repo@skill> -g -y

💡 提示

Star 数量会随时间变化，建议定期查看 skills.sh 获取最新热门 skills。

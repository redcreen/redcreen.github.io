⭐ 最受欢迎的 Skills

ClawHub 官方推荐的热门 Skills

以下是 clawhub.ai/skills 上的热门 Skills，按官方排序：

1

🌤️ weather

获取当前天气和天气预报，支持全球任意地点，无需 API Key。

npx clawhub install weather
2

📝 skill-creator

创建、编辑、改进或审计 Agent Skills，支持 SKILL.md 规范验证。

npx clawhub install skill-creator
3

🔐 healthcheck

主机安全加固和风险评估，支持防火墙、SSH、更新检查等。

npx clawhub install healthcheck
4

🐙 github

GitHub 操作集成，支持仓库管理、Issue 处理、PR 管理等。

npx clawhub install github
5

🚀 vercel-deploy

一键部署到 Vercel，支持静态网站、Next.js、React 等框架。

npx clawhub install vercel-deploy
6

🌐 web-deploy-github

自动部署到 GitHub Pages，适合静态网站和文档站点。

npx clawhub install web-deploy-github
7

📊 kanban

构建多项目看板系统，支持任务管理和进度追踪。

npx clawhub install kanban
8

🔍 github-search

强大的 GitHub 搜索功能，快速查找仓库、代码、Issue。

npx clawhub install github-search
9

🎫 github-issue-resolver

自动分析和解决 GitHub Issue，提供修复建议。

npx clawhub install github-issue-resolver
10

⚡ github-actions-generator

自动生成 GitHub Actions 工作流，支持 CI/CD 场景。

npx clawhub install github-actions-generator

📌 如何安装 Skills

使用 ClawHub CLI 安装：

# 搜索 skills
npx clawhub search <关键词>

# 安装 skill
npx clawhub install <skill-name>

💡 提示

访问 clawhub.ai/skills 发现更多实用 skills，目前共有 50+ 个 skills 可供选择。

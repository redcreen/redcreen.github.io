🔧 OpenClaw 配置文件详解

全面解析 openclaw.json 的每个配置项

配置文件位置

默认路径：~/.openclaw/openclaw.json
环境变量：OPENCLAW_CONFIG_PATH - 可自定义配置文件路径
格式：JSON5（支持注释、尾随逗号）

配置结构概览

完整配置示例

~/.openclaw/openclaw.json 
{
  // ========== 1. 代理配置 ==========
  agents: {
    list: [
      {
        id: "main",
        name: "主助手",
        default: true,
        workspace: "~/.openclaw/workspace",
        agentDir: "~/.openclaw/agents/main/agent",
        model: "moonshot/kimi-k2.5",
        identity: { name: "小助手", emoji: "🤖" },
        sandbox: { mode: "off", scope: "agent" },
        tools: { allow: ["*"], deny: [] }
      },
      {
        id: "work",
        name: "工作助手",
        workspace: "~/.openclaw/workspace-work",
        model: "anthropic/claude-sonnet-4-5",
        sandbox: { mode: "all", scope: "agent" },
        tools: { allow: ["read", "exec"], deny: ["write", "cron"] }
      }
    ]
  },
  
  // ========== 2. 路由绑定配置 ==========
  bindings: [
    { agentId: "main", match: { channel: "whatsapp", accountId: "personal" } },
    { agentId: "work", match: { channel: "whatsapp", accountId: "work" } }
  ],
  
  // ========== 3. 通道配置 ==========
  channels: {
    whatsapp: {
      defaultAccount: "personal",
      dmPolicy: "pairing",
      groupPolicy: "allowlist",
      accounts: {
        personal: { authDir: "~/.openclaw/credentials/whatsapp/personal" },
        work: { authDir: "~/.openclaw/credentials/whatsapp/work" }
      }
    },
    telegram: {
      accounts: {
        default: { botToken: "${TELEGRAM_BOT_TOKEN}", dmPolicy: "pairing" }
      }
    },
    discord: {
      accounts: {
        default: { token: "${DISCORD_BOT_TOKEN}" }
      }
    }
  },
  
  // ========== 4. 工具配置 ==========
  tools: {
    agentToAgent: { enabled: false, allow: ["main", "work"] },
    elevated: { exec: true, write: false }
  },
  
  // ========== 5. 网关配置 ==========
  gateway: {
    host: "127.0.0.1",
    port: 18789,
    token: "${OPENCLAW_GATEWAY_TOKEN}"
  }
}

核心配置项详解

agents（代理配置）

配置项类型必填说明
idstring✓代理唯一标识
namestring✗显示名称
defaultboolean✗是否为默认代理
workspacestring✗工作区路径
agentDirstring✗代理状态目录
modelstring✗默认AI模型

sandbox（沙盒配置）

配置项可选值说明
modeoff / all / ask沙盒模式
scopeagent / shared沙盒范围

tools（工具权限）

配置项说明
allow允许的工具列表，["*"]表示全部
deny禁用的工具列表

bindings（路由绑定）

配置项类型说明
agentIdstring目标代理ID
match.channelstring通道类型(whatsapp/telegram/discord等)
match.accountIdstring账号ID，"*"匹配所有
match.peer.kindstringdirect/group
match.peer.idstring具体ID(手机号/群ID)

channels（通道配置）

配置项可选值说明
dmPolicypairing/allowlist/all私信策略
groupPolicyallowlist/all/none群聊策略
allowFromarray允许列表
defaultAccountstring默认账号

gateway（网关配置）

配置项类型默认值说明
hoststring127.0.0.1绑定地址
portnumber18789绑定端口
tokenstring-访问令牌(可选)

常用配置模板

模板1：单用户基础配置

{
  agents: {
    list: [{ id: "main", default: true }]
  },
  channels: {
    webchat: { enabled: true }
  }
}

模板2：工作/生活分离

{
  agents: {
    list: [
      { id: "personal", name: "个人", default: true },
      { id: "work", name: "工作", sandbox: { mode: "all" } }
    ]
  },
  bindings: [
    { agentId: "personal", match: { channel: "whatsapp", accountId: "personal" } },
    { agentId: "work", match: { channel: "whatsapp", accountId: "work" } }
  ]
}

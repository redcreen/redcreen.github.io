🚀 什么是 OpenClaw？

OpenClaw 是一个 AI 助手平台，让你可以在多个聊天渠道（Telegram、WhatsApp、Slack 等）与 AI 助手对话，并执行各种任务。

  • 💬 支持多种聊天渠道
  • 🤖 多 Agent 并行工作
  • 🔌 丰富的 Skills 扩展
  • ⚡ 本地优先，数据自主

🛠️ 安装与启动

1. 安装

bash 
# macOS / Linux
brew install openclaw

# 或使用安装脚本
curl -fsSL https://openclaw.dev/install.sh | sh

2. 初始化配置

bash 
# 创建配置文件
openclaw init

# 编辑配置
openclaw config edit

3. 启动 Gateway

bash 
# 启动服务
openclaw gateway start

# 或后台运行
openclaw gateway start --daemon

📚 核心概念

🤖

Agent

AI 助手实例，每个 Agent 有独立的配置、记忆和工具权限

Gateway

消息网关，负责接收各渠道消息并路由给对应 Agent

聊天渠道，如 Telegram、WhatsApp、Slack 等

🔌

Skill

功能扩展，让 Agent 获得新能力，如查天气、发邮件等

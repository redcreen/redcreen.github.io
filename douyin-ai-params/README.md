# 抖音视频分析归档

本目录包含该视频的原始文件、截图、清洗后的逐字稿与摘要文档。

## 文件说明

- `douyin_merged.mp4`：合并后的完整视频
- `douyin_audio_16k.wav`：转写用音频
- `douyin_transcript.txt`：原始自动转写文本
- `douyin_transcript.md`：原始转写 Markdown
- `douyin_transcript_clean.md`：清洗校对后的逐字稿
- `douyin_summary.md`：结构化摘要
- `douyin_report_with_images.md`：带截图引用的阅读版文档
- `images/`：文档中引用的截图
